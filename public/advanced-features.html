    Creating Custom Plugins

    Example: Extract tags from front matter

    #!/bin/sh
# plugins/extract_tags.sh
set -e

MARKDOWN_DIR="${MARKDOWN_DIR:-./content}"
HTML_DIR="${HTML_DIR:-./public}"

for md in "$MARKDOWN_DIR"/*.md; do
  [ -e "$md" ] || continue
  
  tags=$(awk '
    BEGIN{inmeta=0}
    /^---$/{ if(inmeta==0){inmeta=1;next}else{exit} }
    inmeta==1 && $0 ~ "^tags:" {
      sub("^tags:[ ]*", ""); print; exit
    }' "$md")
  
  if [ -n "$tags" ]; then
    echo "$(basename "$md"): $tags"
  fi
done

    Plugin Best Practices

    1. Exit gracefully if dependencies missing
    2. Use environment variables for paths
    3. Print clear log messages
    4. Be idempotent (safe to run multiple times)
    5. Handle errors without breaking the build

    Watch Mode

    Live rebuild on file changes using entr:

    ./bin/cruella watch

    How It Works

    1. Monitors content/*.md files
    2. Triggers build_all.sh on change
    3. Logs rebuild timestamp
    4. Continues watching

    Tips

    • Use with serve.sh for live preview
    • Add content/**/*.md to watch subdirectories (future enhancement)
    • Restart watch if you add new files

    Deployment Options

    GitHub Pages

    # .github/workflows/deploy.yml
name: Deploy
on:
  push:
    branches: [main]
jobs:
  build-deploy:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v3
      - name: Build
        run: |
          sudo apt-get install -y pandoc jq
          ./bin/cruella build html
      - name: Deploy
        uses: peaceiris/actions-gh-pages@v3
        with:
          github_token: ${{ secrets.GITHUB_TOKEN }}
          publish_dir: ./public

    Troubleshooting

    Common Issues

    Pandoc Not Found

    # Ubuntu/Debian
sudo apt-get install pandoc

# macOS
brew install pandoc

    LaTeX Errors in PDF

    # Install TeX Live
sudo apt-get install texlive-xetex

# Or use minimal engine
pandoc --pdf-engine=pdflatex input.md -o output.pdf

    Watch Mode Not Working

    # Install entr
sudo apt-get install entr

# Or macOS
brew install entr

    Search Not Working

    # Install jq
sudo apt-get install jq

# Rebuild to generate index
./bin/cruella build html
